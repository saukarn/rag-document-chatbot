# 📚 RAG Document Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built with:

- langchain-classic
- OpenAI
- ChromaDB
- Streamlit

## Features
- Upload PDFs
- Ask questions from documents
- Source-based answers

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
